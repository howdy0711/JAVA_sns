1. 회원가입을 하면 db에 회원정보가 등록
2. 가입된 id로 로그인
3. 서버는 기본적으로 멀티 스레드 서버로 클라이언트가 요청을하게되면 새로운 소캣이생성됨
      클라이언트별로 서버에 디렉토리가 구성되어있음
4. 서버의 포트번호를 다르게하여, 사진을 전송하는 스트림과, 채팅 (텍스트를) 전송하는
스트림이 2개있음
5. 프로필 사진을 바꾸게되면 자신의 로컬내의있는 사진이 서버로 전송되고 서버는 그에맞는
클라이언트 디렉토리에 프로필사진을 추가시키게 되므로, 어느컴퓨터에서나 같은 프로필사진을
볼수 있게된다.
6. 
7. 공부하기 버튼을 누르게되면, 서버에 저장되어있는 메모장파일의 정리된 ccna데이터들을
읽어서 각 주제에 맞는 파일들이 텍스트 area에 더해진다.
8. 시험보기 버튼을 누르게되면, 서버에 저장되어있는 문제파일을 파일 입출력으로 읽어.
문제를 보여준다. 시험이끝나게되면 정답을 적어놓은 메모장파일과 본인이 선택한 문자열을
비교하여 자신의 점수를 알려주게된다.
9. 사진첩 버튼을 누르게되면 서버에 등록된 사진앨범을 볼수있다. (카드레이아웃으로 구성됨)
10. 시간표 버튼을 누르면 자신의 시간표를 등록, 수정, 저장할수있다. 
11. 타임라인에서는 등록되는 정보는 서버에서 모든 클라이언트에게 보내어 접속한 사용자들이
같은 타임라인을 볼수 있게된다.
12. 친구목록은 db에 기록된 사용자들이 접속여부를 체크하여 접속한 사용자는 초록색 사각형으로
표시된다.

서버에는 각서비스별로 소켓을 만들어서 각각의 다른서비스를한다.

1번 채팅서버
2번 타임라인
3번 주서버 
4번 앨범

서버에서는 이러한정보를 해당 클라이언트 디렉토리마다 메모장으로 저장한다.  
그리고 서버는 모든 클라이언트에게 타임라인 정보를 넘겨준다.
